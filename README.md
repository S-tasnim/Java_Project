# Java Stopwatch

## Description

This project is a Java-based Stopwatch application created using the Swing framework. It was developed as part of a university coursework assignment to gain hands-on experience with graphical user interfaces (GUI), event-driven programming, and Java's Timer class. The application accurately tracks elapsed time and provides an intuitive interface with Start, Stop, and Reset controls.

## Features

- Start and stop the stopwatch
- Reset the timer to **00:00:00**
- Display elapsed time in **HH:MM:SS** format
- Built with Java Swing
- Uses `ActionListener` and `Timer` for event handling and real-time updates

## Technologies Used

- Java
- Java Swing
- AWT
- `javax.swing.Timer`
